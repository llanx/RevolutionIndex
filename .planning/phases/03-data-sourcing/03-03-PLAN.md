**Domain 4: Social Mobilization & Trust (11 variables):**

1. **#7 Government Trust / State Legitimacy** (Strong, other-data)
   - Primary: Pew Research Center "Public Trust in Government: 1958-2025" (published reports with data tables)
   - Alternative: ANES trust-in-government series from cumulative file (VCF0604, biennial/quadrennial)
   - Alternative: Gallup institutional confidence surveys (1973-present, published reports)
   - Document: All three are periodic surveys with irregular spacing -- not continuous time series
   - Access method: Manual extraction from published reports/data tables (Pew, Gallup) or CSV download (ANES)
   - Recommend: ANES as primary (most standardized methodology), Pew as supplementary (higher frequency)

2. **#12 Protest Frequency and Participation** (Strong, other-data)
   - Primary: ACLED US event data (API with free registration from acleddata.com, 2020-present, 30K+ events)
   - Document: API endpoint, event type filters (protests, riots, demonstrations), free registration required
   - Alternative: NAVCO dataset for historical protest campaigns (but aggregated, not event-level)
   - Alternative: Crowd Counting Consortium (manual data collection of US protests)
   - Tag as "short series" (2020-present only for real-time data)
   - Note: Pre-2020 US protest data requires historical compilation from news archives or Global Database of Events (GDELT)

3. **#25 Civil Society Density / Union Membership** (Moderate, fed-data)
   - Primary: BLS union membership annual release (union2.htm, 1983-present)
   - Use `bls_search_series` to search for union membership series
   - Alternative: BLS Current Population Survey (CPS) union data
   - Access method: Manual extraction from annual news release tables
   - Document: National + state level, annual frequency, 1983-present for current methodology

4. **#26 Youth Unemployment / Disconnection** (Moderate, fed-data)
   - Primary: SLUEM1524ZSUSA on FRED (youth unemployment 15-24, annual, 1991-present)
   - Use `fred_series_info` to verify SLUEM1524ZSUSA
   - Alternative: BLS LNS series (LNS14000012 etc.) for monthly US youth unemployment
   - Use `bls_search_series` for youth unemployment
   - Alternative: Census ACS "disconnected youth" (NEET: not in education, employment, or training)

5. **#30 Democratic Commitment (Attitudinal)** (Moderate, other-data)
   - Primary: ANES cumulative file -- support for democratic norms questions
   - Alternative: World Values Survey (WVS) -- "importance of living in a democracy" question (US waves)
   - Alternative: Bright Line Watch expert surveys on democratic norm adherence (2017-present)
   - Note: Foa & Mounk (2016) and Mounk (2018) used WVS data to argue declining democratic commitment
   - Irregular frequency across all sources

6. **#34 Conspiratorial Thinking Prevalence** (Weak, other-data)

   - Alternative: Uscinski & Parent (2014) survey data (historical, one-time)
   - Alternative: Pew surveys on conspiracy beliefs (irregular)
   - Note: Weak-rated variable -- include as cataloged per locked decision but flag evidence limitations

7. **#36 Protest Diffusion / Contagion** (Weak, other-data)
   - Primary: Constructible from ACLED event data (spatial/temporal clustering of protest events)
   - Document construction recipe: Geographic and temporal proximity analysis of ACLED protest events
   - Note: Requires ACLED data from #12 plus geospatial analysis -- a derived measure, not a raw data source

8. **#37 Prior Protest Experience** (Weak, other-data)
   - Primary: Constructible from ACLED cumulative event counts by region/city
   - Alternative: Historical protest catalogs (pre-2020) from academic compilations
   - Note: Weak-rated, derived from same ACLED data as #12 and #36

9. **#41 Institutional Legitimacy Denial** (Weak, other-data)

   - Alternative: Post-2020 election denial surveys (Pew, various academic surveys)

10. **#42 Political Efficacy Beliefs** (Weak, other-data)
    - Primary: ANES cumulative file -- internal and external political efficacy questions
    - Specific variables: VCF0613 (external efficacy), VCF0614 (internal efficacy)
    - Note: Biennial/quadrennial frequency, 1952-present

11. **#44 Cross-Class Coalition Formation** (Weak, unknown)
    - Classification: Unavailable -- no direct measure or strong proxy
    - Document the gap: Theoretically important (Wickham-Crowley 1992, Foran 2005) but no standardized measurement exists for cross-class political coalition breadth in the US
    - Nearest proxy: Union membership trends (#25) as very rough indicator of cross-class organizing, but this is NOT a strong proxy (does not meet the literature-validated mapping requirement)
    - Flag for future work

**Domain 5: Information / Media Ecosystem (4 variables):**

12. **#28 Media Trust / Partisan Media Trust Gap** (Moderate, other-data)
    - Primary: Gallup "Confidence in Mass Media" (1973-present, annual poll)
    - Alternative: Pew media trust surveys (partisan breakdown, 2014-present)
    - Alternative: Reuters Institute Digital News Report (annual, 2012-present, US included)
    - Access: Published reports with data tables; no API
    - Document: Partisan media trust GAP (difference between D and R trust levels) is the more theoretically relevant measure than aggregate trust

13. **#33 Misinformation Prevalence / Exposure** (Weak, Contested, unknown)
    - Classification: Unavailable -- no standardized, regularly-updated measure of misinformation exposure exists
    - Document the gap: Vosoughi et al. (2018) demonstrated false news spreads faster on Twitter but did not create a replicable time series. No ongoing measurement program exists.
    - Nearest potential: Meta/platform transparency reports (irregular, limited access, not comparable over time)
    - Flag for future work

14. **#35 Social Media Political Engagement** (Weak, Contested, unknown)
    - Classification: Unavailable -- no standardized, freely-available time series of political social media engagement
    - Document the gap: Tufekci (2017) and Gonzalez-Bailon (2011) studied mechanisms but no ongoing US measurement exists
    - Nearest potential: Pew social media surveys (irregular), platform-specific APIs (restrictive, not free)
    - Flag for future work

15. **#43 Information Fragmentation / Echo Chambers** (Weak, Contested, unknown)
    - Classification: Unavailable -- no standardized measure of information ecosystem fragmentation
    - Document the gap: Sunstein (2001) and Bail et al. (2018) studied echo chambers but disagreed on magnitude. No replicable index exists.
    - Flag for future work

**Section 1: Availability Summary Matrix**

Create a single table covering all 45 variables with columns:
| # | Variable | Evidence Rating | Availability | Primary Source | Frequency | Coverage | Short Series? |

Where:
- # matches the Phase 2 catalog number (1-45)
- Evidence Rating from catalog (Strong/Moderate/Weak + Contested if applicable)
- Availability uses the 4-tier taxonomy: Available (free API), Available (manual download), Partially available (proxy needed), Unavailable
- Primary Source: the recommended data source name
- Frequency: native data frequency (daily/weekly/monthly/quarterly/annual/irregular)
- Coverage: start year to present (or end year)
- Short Series: Yes if data starts after 2000

**Count verification:** After building the matrix, tally and report:
- Total: 45 variables
- Available (free API): count (expected ~20 from FRED/BLS/BEA/Census/Treasury MCP sources)
- Available (manual download): count (expected ~12 from V-Dem, VoteView, WID, ANES, Pew, Gallup, ACLED)
- Partially available (proxy needed): count (expected ~6 from constructed measures)
- Unavailable: count (expected ~4 from info/media + cross-class coalition)

Verify counts sum to 45. If not, find the discrepancy and fix.

**Section 2: Gap Analysis**

List all variables classified as "Unavailable" or "Partially available (proxy needed)":

For each unavailable variable:
- Variable name and catalog number
- Evidence rating and theoretical importance
- Why unavailable (no standardized measure, no US source, paywalled, etc.)
- Nearest potential proxy and why it doesn't qualify as "strong" per locked decision
- Priority for future work (High/Medium/Low based on evidence rating and theoretical centrality)

For each partially available variable:
- Variable name and catalog number
- What is available and what is missing
- Construction recipe for the proxy measure
- Confidence in proxy validity (from literature support)

- URL: Primary access URL
- API Key Required: Yes (free) / Yes (paid) / No
- Rate Limits: Known limits or "Unknown"
- License: Terms of use
