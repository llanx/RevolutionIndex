**1. models/models.py -- Shared types and model registry:**

```python
"""
Shared model output types and model registry.
All models return ModelOutput. The registry enables iteration over all models.
"""
from dataclasses import dataclass, field
from typing import Callable
import pandas as pd

@dataclass
class ComponentScore:
    """A named sub-score within a model."""
    name: str
    value: float          # 0.0-1.0
    variables_used: list[str] = field(default_factory=list)

@dataclass
class ModelOutput:
    """Standardized output from any model's compute function."""
    model_id: str
    model_name: str
    score: float          # 0.0-100.0
    components: list[ComponentScore] = field(default_factory=list)
    domain_contributions: dict[str, float] = field(default_factory=dict)
    timestamp: str = ""
    variables_used: list[int] = field(default_factory=list)

# Model function signature: (DataFrame) -> ModelOutput
ModelFn = Callable[[pd.DataFrame], ModelOutput]

# Registry populated by model files
MODEL_REGISTRY: dict[str, ModelFn] = {}

def register_model(model_id: str):
    """Decorator to register a model function."""
    def decorator(fn: ModelFn) -> ModelFn:
        MODEL_REGISTRY[model_id] = fn
        return fn
    return decorator
```

**2. models/model_psi.py -- Turchin PSI with Phase 1 Math Fixes:**

Read `validation/model-assessment-turchin-psi.md` and `validation/math-fix-checklist.md` for the specific fixes to apply.

Structure: PSI = MMP x EMP x SFD (multiplicative, meaning instability requires all three pressures).

**Phase 1 math fixes to apply:**
- **Geometric mean instead of arithmetic:** Use `(MMP * EMP * SFD) ** (1/3)` for the composite, not `(MMP + EMP + SFD) / 3`. The geometric mean preserves the multiplicative interaction that Turchin's theory requires.
- **Rolling z-score normalization instead of min-max:** Each component uses rolling z-score -> CDF mapping (from normalize.py). This fixes the pinning bug (impl A1) where min-max normalization pinned 2 of 3 PSI components near 1.0.
- **Labor share series:** Use PRS85006173 (nonfarm business sector labor share), recommended in Phase 3 inventory over W270RE1A156NBEA.

Component mappings (from variable catalog to model components):
- **MMP (Mass Mobilization Potential):** Variable #2 (labor share, inverted: lower share = more stress), Variable #9 (unemployment), Variable #26 (youth unemployment)
- **EMP (Elite Mobilization Potential):** Variable #8 (elite overproduction via education-job mismatch or WID top 1%), Variable #11 (elite factionalism), Variable #19 (intra-elite wealth gap), Variable #45 (wealth concentration top 0.1%)
- **SFD (State Fiscal Distress):** Variable #5 (debt/deficit), Variable #6 (financial stress), Variable #7 (government trust, inverted)

Each component: evidence-weighted average of its constituent variables, then combined via geometric mean.

```python
def compute_psi(unified_df: pd.DataFrame) -> ModelOutput:
    """
    Turchin Structural-Demographic PSI.
    PSI = geometric_mean(MMP, EMP, SFD) * 100

    Phase 1 fixes applied:
    - Geometric mean (not arithmetic) for composite
    - Rolling z-score normalization (not min-max)
    - PRS85006173 for labor share
    """
```

Score: geometric mean of 3 components * 100, clamped to 0-100.

**1. models/model_georgescu.py -- Georgescu SDT for Industrialized Societies:**

Read `literature/framework-assessment.md` section "8. Georgescu SDT for Industrialized Societies" for the theoretical basis and required inputs.

Georgescu (2023) operationalizes structural-demographic theory for modern developed economies with three components that parallel Turchin's PSI but use different, US-specific proxies:

- **Elite Overproduction (modern proxy):** Education-job mismatch ratio. Variable #8 (elite overproduction, constructed from Census advanced degrees / BLS professional job openings). This is the key differentiator from PSI, which uses income concentration. Phase 2 identified this as "potentially better EMP proxy than top 1% income share for US context."

- **Mass Immiseration (modern proxy):** Cost-of-living adjusted wage stagnation. Variable #2 (real wage/labor share), Variable #16 (housing affordability, inverted), Variable #17 (inflation), Variable #40 (cost of living composite). Georgescu argues that in industrialized societies, immiseration manifests through cost-of-living pressure rather than subsistence crisis.

- **State Fiscal Distress:** Variable #5 (debt/deficit), same conceptual dimension as PSI-SFD but normalized independently.

Structure: Weighted combination of three components (not necessarily multiplicative like PSI -- Georgescu's empirical approach uses correlation-based weighting rather than strict multiplicative interaction).

```python
def compute_georgescu(unified_df: pd.DataFrame) -> ModelOutput:
    """
    Georgescu SDT for Industrialized Societies.
    Operationalizes structural-demographic theory with modern proxies:
    - Education-job mismatch for elite overproduction
    - Cost-of-living adjusted wages for mass immiseration
    - Government debt trajectory for fiscal distress

    Unlike PSI, uses weighted average (not multiplicative) per Georgescu's
    empirical approach for developed economies.
    """
```

Component weights: Elite Overproduction 0.35 (strongest correlation in Georgescu 2023), Mass Immiseration 0.40 (most variable in developed economies), State Fiscal Distress 0.25 (persistent but slow-moving).

Score: weighted average of 3 components * 100, clamped to 0-100.

**2. models/model_vdem.py -- V-Dem ERT Institutional Quality Model:**

Read `literature/framework-assessment.md` section "4. V-Dem Episodes of Regime Transformation" for the theoretical basis.

V-Dem provides continuous democratic quality measures. The key insight from Phase 2 is that for the US (near the ceiling on absolute V-Dem scores), **rate-of-change analysis is more informative than absolute levels.** The US Liberal Democracy Index declined from ~0.89 to ~0.72 (2015-2022), which is significant even though the absolute level remains "high" by global standards.

Components (all from V-Dem indicators, annual, manual download):
- **Liberal Democracy trajectory:** Variable #13 (v2x_libdem). Rate of change over rolling 5-year window.
- **Judicial Independence trajectory:** Variable #21 (v2x_jucon). Rate of change.
- **Freedom of Expression trajectory:** Variable #22 (v2x_freexp_altinf). Rate of change.
- **Legislative Constraints trajectory:** Variable #23 (v2xlg_legcon). Rate of change.
- **Electoral Integrity trajectory:** Variable #24 (v2xel_frefair). Rate of change.
- **Executive Aggrandizement trajectory:** Variable #32 (v2exrescon). Rate of change (inverted: decline = more stress).

Rate-of-change methodology:
- For each V-Dem indicator: compute 5-year rolling difference (current year minus 5 years ago)
- Negative change = deterioration = higher stress
- Map the rate of change to 0.0-1.0 using the historical distribution of changes across V-Dem's full 1789-present dataset for the US
- Weight by democratic dimension importance: Liberal Democracy 0.25, Judicial 0.20, Expression 0.15, Legislative 0.15, Electoral 0.15, Executive 0.10

Additionally, include:
- **State Capacity:** Variable #38 (World Bank WGI GE.EST). Level, not rate of change.
- **Voter Access:** Variable #29 (Grumbach SDI). Level.

```python
def compute_vdem(unified_df: pd.DataFrame) -> ModelOutput:

    Key insight: For near-ceiling US scores, trends matter more than levels.
```
