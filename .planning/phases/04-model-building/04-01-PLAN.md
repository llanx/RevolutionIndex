**1. models/README.md -- Architecture Selection Document (MOD-01)**

Document the architecture selection with explicit rationale tying to literature findings and data availability:

- **Architecture:** 5-model evidence-weighted ensemble producing a 0-100 composite score
- **Models selected:** (1) Turchin PSI with Phase 1 math fixes (geometric mean, corrected normalization), (2) Prospect Theory PLI with K constant corrections, (3) Financial Stress Pathway with CSCICP03USM665S weight redistribution, (4) Georgescu SDT for industrialized societies (education-job mismatch proxy), (5) V-Dem ERT institutional quality tracking (rate-of-change analysis)
- **Why this architecture:** Literature mining (Phase 2) found existing 3 models cover only 27% of 45 discovered variables. Georgescu SDT covers the elite overproduction dimension with better US-applicable proxies. V-Dem ERT covers the institutional/democratic quality dimension that was entirely uncovered. Together, 5 models span all 5 identified domains.
- **Ensemble rationale:** Evidence-weighted average. Initial weights: PSI 0.25 (strong evidence, long validation history), PLI 0.20 (strong economic evidence, prospect theory well-validated), FSP 0.15 (transmission mechanism well-documented but narrower scope), Georgescu SDT 0.25 (most directly applicable to developed democracies), V-Dem ERT 0.15 (primarily diagnostic, not predictive, but captures institutional dimension). Note: these are provisional, Phase 5 validation may adjust.
- **Data availability constraint:** 41 of 45 variables measurable (91%). 4 unavailable variables (all weak-rated) are dropped. 6 proxy-needed variables have documented construction recipes.
- **Factor grouping:** 5 domains from Phase 3 inventory serve as factor groups: economic_stress (13 vars), political_polarization (8 vars), institutional_quality (8 vars), social_mobilization (11 vars), information_media (4 vars, reduced to 1 after dropping 3 unavailable).
- **Score calibration approach:** Historical anchor points. 2008 financial crisis and 2020 should score in Crisis Territory (51-75). Mid-1990s should score in Stable (0-25).

Normalization functions for mapping raw variable values to 0.0-1.0 stress intensity.

Uses rolling z-scores (recommended by Phase 2 literature review for trending US
macroeconomic series) as the primary approach, with percentile-rank fallback.
"""
import numpy as np
import pandas as pd
from scipy import stats

def rolling_zscore(series: pd.Series, window: int = 240) -> pd.Series:
    """
    Rolling z-score normalization (recommended over min-max for trending series).

    Window default: 240 months (20 years) to capture secular trends while
    remaining sensitive to recent changes.

    Returns z-scores (unbounded). Use zscore_to_stress() to map to 0.0-1.0.
    """

def zscore_to_stress(z: float, direction: str = "higher_is_worse") -> float:
    """
    Map a z-score to 0.0-1.0 stress intensity using the standard normal CDF.

    For higher_is_worse: stress = CDF(z) -> z=0 maps to 0.5, z=2 maps to ~0.977
    For lower_is_worse: stress = 1 - CDF(z) -> z=0 maps to 0.5, z=-2 maps to ~0.977
    """

def percentile_rank(series: pd.Series) -> pd.Series:
    """
    Percentile-rank normalization. Fallback for series too short for rolling z-score.
    Maps each observation to its rank within the full history, scaled to 0.0-1.0.
    """

def normalize_variable(raw_series: pd.Series, direction: str,
                       method: str = "rolling_zscore", window: int = 240) -> pd.Series:

    Main normalization entry point. Applies rolling z-score by default,
    falls back to percentile rank for short series (< window observations).

    Returns: pd.Series with values in 0.0-1.0 range (stress intensity).
    """

Implement all four functions. Key design decisions:
- Rolling z-score with 20-year window is the default (Phase 2 recommendation: rolling z-scores over min-max for trending US macroeconomic series)
- Standard normal CDF maps z-scores to 0.0-1.0 (avoids the min-max pinning bug identified in Phase 1 for PSI)
- For short series (< window observations), fall back to percentile rank over the available history
- Direction parameter flips the mapping so that "higher raw value = higher stress" vs "lower raw value = higher stress" is handled correctly
